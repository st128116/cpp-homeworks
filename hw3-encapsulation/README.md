# Домашнее задание №3 — Инкапсуляция

## Условие
Реализовать классы `BankAccount` и `Bank` с использованием принципов
объектно-ориентированного программирования и инкапсуляции.

## Реализация
- Класс `BankAccount` хранит баланс счёта и предоставляет методы для его изменения.
- Класс `Bank` управляет набором банковских счетов и выполняет операции над ними.
- Прямой доступ к данным ограничен.
