# HW8 — Template Array

Шаблонный класс `Array<T>` с фиксированным размером.

## Возможности
- хранение элементов шаблонного типа
- вывод элементов
- поиск минимального значения между двумя массивами

## Структура
- `src/main.cpp` — реализация
- `Makefile` — сборка

## Сборка и запуск

```bash
make
./hw8_array
