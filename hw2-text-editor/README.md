# HW2 — Text Editor with Cursor

Текстовый редактор с курсором.

Поддерживаемые операции:
- добавление текста в позицию курсора
- удаление символов слева от курсора
- перемещение курсора влево и вправо

## Структура
- `src/main.cpp` — реализация
- `Makefile` — сборка проекта

## Сборка и запуск

```bash
make
./text_editor
