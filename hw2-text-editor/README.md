# Text Editor with Cursor

Домашнее задание №2.

Реализация текстового редактора с курсором:
- добавление текста
- удаление (Backspace)
- перемещение курсора влево и вправо

## Реализация
Используется два стека:
- левый стек — символы слева от курсора
- правый стек — символы справа от курсора

## Сборка и запуск

```bash
g++ text_editor.cpp -std=c++17 -o editor
./editor
