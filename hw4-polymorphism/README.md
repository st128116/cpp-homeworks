# Домашнее задание №4 — Полиморфизм

## Условие
Реализовать оператор `[]` для пользовательского класса `String` таким образом,
чтобы выражение `str[i][j]` возвращало подстроку, начинающуюся в позиции `i`
(включительно) и заканчивающуюся в позиции `j` (не включая).

## Реализация
- Оператор `[]` реализован с использованием вспомогательного proxy-класса.
- Результатом является новый объект класса `String`.
- Гарантируется, что индексы не выходят за границы строки.

## Пример использования
```cpp
String s("hello");
String sub = s[1][4]; // "ell"
